Descripción del Proyecto

Este sitio fue desarrollado para la Escuela de Innovación (ESIM) como landing institucional. Utilicé una plantilla HTML que personalicé completamente con colores institucionales, fuentes y distribución de contenido. Se actualiza semanalmente con actividades escolares, noticias destacadas, información para ingresantes y secciones específicas como Profesorado, Tramo de Formación Pedagógica, Bachilleres, etc. Todo el contenido es 100% responsive y optimizado para visibilidad en buscadores.
